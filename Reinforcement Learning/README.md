## GitHub Repositories

### Core Reinforcement Learning Frameworks

| Repository | Description | Key Features |
| :--- | :--- | :--- |
| **[Learn Deep Reinforcement Learning in 60 days\!](https://github.com/andri27-ts/Reinforcement-Learning?tab=readme-ov-file)** | A structured Deep RL course with lectures and full code in **Python/PyTorch**. Great for beginners seeking a clear path. | **DQN, PPO, A2C, DDPG, Model-Based RL**, tested on Atari and RoboSchool. |
| **[Implementation of Reinforcement Learning Algorithms](https://github.com/dennybritz/reinforcement-learning)** | Contains code implementations and exercises designed to accompany **Sutton & Barto's** definitive RL textbook and **David Silver's** lecture series. | Perfect for academic/theoretical learning; implementations in **Python/TensorFlow**. |
| **[Modularized Implementation of Deep RL Algorithms in PyTorch](https://github.com/ShangtongZhang/DeepRL)** | A clean, modular library for implementing various modern Deep RL algorithms. | **PyTorch**-centric, focused on modern, clean code structure. |
| **[Minimal and Clean Reinforcement Learning Examples](https://github.com/rlcode/reinforcement-learning)** | Highly readable implementations of classic RL algorithms, ideal for quickly understanding concepts. | Focus on clarity and simplicity. |
| **[PyTorch implementation of reinforcement learning algorithms](https://github.com/Khrylx/PyTorch-RL)** | Another quality repository providing clear implementations of RL algorithms using PyTorch. | Good alternative or supplementary resource for PyTorch users. |

### AI Agent Tools & Architectures

| Repository | Description | Key Features |
| :--- | :--- | :--- |
| **[Airweave: Context Retrieval for AI Agents across Apps & Databases](https://github.com/airweave-ai/airweave)** | An open-source **Retrieval-Augmented Generation (RAG)** layer that turns external data sources (like GitHub, Notion, Stripe) into a searchable knowledge base for your agents. | Context retrieval via **REST API** or **MCP**, data synchronization, semantic search. |
| **[FastMCP v2](https://github.com/jlowin/fastmcp)** | A fast, Pythonic library for easily building and interacting with **Model Context Protocol (MCP)** servers and clients. | Streamlines agent-to-service communication. |
| **[Agentic AI Crash Course](https://github.com/aishwaryanr/awesome-generative-ai-guide/tree/main/free_courses/agentic_ai_crash_course)** | A collection of resources and materials for learning the fundamentals of agentic AI design. | Curriculum and links for getting started with agents. |
| **[Verifiers: Environments for LLM Reinforcement Learning](https://github.com/PrimeIntellect-ai/verifiers)** | Specialized repository providing environments for applying RL to train LLMs (e.g., via **RLHF** or RL from AI feedback). | Focuses on the LLM-specific application of RL. |
